
## Overview

This project analyzes **NIFTY 50 historical market data** and implements a **simple rule-based trading strategy** using Python.
It focuses on **data cleaning, return calculation, strategy backtesting, and performance evaluation** while avoiding common backtesting mistakes such as look-ahead bias.

---

## Dataset

* **Market:** NIFTY 50 (NSE India)
* **Frequency:** Daily data
* **Source:**

  * Kaggle: [https://www.kaggle.com/datasets/rohanrao/nifty50-stock-market-data](https://www.kaggle.com/datasets/rohanrao/nifty50-stock-market-data)

What This Project Does

* Cleans and processes time-series market data using Pandas
* Calculates:

  * Daily returns
  * Rolling volatility
  * Cumulative P&L
  * Drawdowns
* Implements a simple moving-average based trading strategy
* Evaluates performance using **Sharpe Ratio**
* Visualizes equity curve and drawdown
  
Tech Stack
* Python
* Pandas
* NumPy
* Matplotlib
  
Key Learning

* Time-series data handling
* Basic financial metrics
* Rule-based strategy backtesting
* Avoiding look-ahead bias
